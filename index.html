Standard Template

A template for consistent records—same info, same place, every time.

  • Lead image
  • Artist — name; short description (nationality, birth year)
  • Title — work title
  • Year
  • Short description — 1–2 sentences
  • Credit line
  • Accession number
  • Curatorial Description — interpretation & significance
  • Technical Narrative — how it works
  • Components — equipment, media, parts
  • Exhibitions
  • Installation — room/setup requirements
  • Iterations — versions/updates
  • Manuals & Hardware Information
  • References — citations & related docs

Mini‑Lab (30 min)

  1. Create page: Artwork:Jim Campbell/Title (Year)
  2. Paste the skeleton
  3. Fill the table (Artist, Title, Year, Credit, Accession)
  4. Add ≥ 2 images to the gallery
  5. Write Curatorial Description
  6. Add Categories: Media art records, Jim Campbell, 20XX
  7. Save with a clear Edit Summary
Encourage VisualEditor first; demonstrate switching to source only for the table or gallery tweaks.
